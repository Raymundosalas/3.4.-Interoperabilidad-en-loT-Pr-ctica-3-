# README - Conexión a Internet y Envío de Datos

Este proyecto fue desarrollado en Unity con Firebase para realizar conexión a internet y enviar datos a la nube.

## Funciones principales

- Conectar Unity con Firebase.
- Guardar datos del jugador.
- Leer información almacenada.
- Actualizar registros.
- Eliminar datos.

## Datos enviados

- Nombre
- Puntos
- Nivel
- Monedas

## Tecnologías utilizadas

- Unity
- C#
- Firebase Realtime Database

## Resultado

Se logró establecer conexión correctamente y enviar datos a la base de datos en la nube.
